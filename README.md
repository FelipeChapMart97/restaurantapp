# RestaurantApp - Menú Virtual

## Descripción
RestaurantApp es una aplicación web de menú virtual interactivo que permite a los clientes de restaurantes acceder al menú, realizar pedidos y gestionar su experiencia gastronómica mediante un código QR.

## Características
- Menú digital categorizado
- Sistema de carrito de compras
- Diseño responsivo
- Integración con código QR
- Formulario de contacto
- Despliegue automatizado

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript Vanilla
- GitHub Pages (Hosting)
- Vercel (Deploy automático)
